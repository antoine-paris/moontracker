# Social Media Images for SpaceView.me

This folder should contain the following social media images for SEO and sharing:

## Required Images:

### 1. og-image.jpg (1200x630px)
- Open Graph image for Facebook, LinkedIn, etc.
- Should showcase SpaceView interface with moon/planets
- High contrast, readable text
- Include SpaceView.me branding

### 2. twitter-card.jpg (1200x600px) 
- Twitter card summary image
- Similar content to OG image but optimized for Twitter
- Clear, compelling visual of the astronomical simulator

### 3. Screenshots for PWA manifest:
- desktop-main.jpg (1280x720px) - Desktop interface screenshot
- mobile-main.jpg (390x844px) - Mobile interface screenshot

## Design Guidelines:
- Use the SpaceView color scheme (dark backgrounds, bright objects)
- Include the SpaceView.me logo
- Show realistic moon phases, planets, or eclipse simulations
- Ensure text is readable at small sizes
- High quality, professional appearance

## Alternative Text:
All images should have descriptive alt text for accessibility:
- "SpaceView.me astronomical simulator showing moon phases and sky visualization"

## File Locations:
- Social sharing images: /public/
- PWA screenshots: /public/screenshots/

These images are referenced in:
- index.html (meta tags)
- manifest.webmanifest (screenshots)
- Various page components
